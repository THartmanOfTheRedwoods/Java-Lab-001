# English Vs. Computer Science (Fight!)

1. In computer jargon, what's the difference between a statement and a commment?
 
  A comment gives us information about what the code is doing or has to do in a computer program, while a statement tells code what to do and commands a task.
  
 2. What does it mean to say that a program is portable? 
 
  A program is portable if it is able to run on more than one kind of computer. 
    
 3. In common English, what does the word compile mean?
 
  To compile means to produce something by putting together information from different sources. 
  
 4. What is an executable? Why is that word used as a noun? 

  An example of something being executable or is executable : A computer file with a program or apart of one that can be executed or used. Executable is a noun because executable is describing something or referring to something, in this case a computer file with a program that is executable.
