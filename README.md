# Exercise 1.1

1. In computer jargon, what’s the difference between a statement and a comment?
* In computer jargon, a **statement** is part of a program that specifies one step of an algorithm, while a **comment** is a part of a program that contains information about the program but has no effect when the program runs.
2. What does it mean to say that a program is portable?
* A **portable program** is a program that can run on different kinds of computers (different kinds of operating systems) with no or little modifications.
3. In common English, what does the word compile mean?
* In common English, **compile** means to collect information from different places and arrange it in a list, report, etc.
4. What is an executable? Why is that word used as a noun?
* An **executable** is a program translated from a high-level program. Executable is used as a noun because it refers to object code that is ready to run on specific hardware.
